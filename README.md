# 🐱 ЛавкинКот - Сервис доставки продуктов

Современное веб-приложение для доставки продуктов с панелями управления для администраторов, партнеров и курьеров.

## 🚀 Технологии

- **Next.js 15** - React фреймворк с App Router
- **TypeScript** - Типизированный JavaScript
- **Lucide React** - Современные иконки
- **Context API** - Управление состоянием
- **CSS-in-JS** - Inline стилизация

## ✨ Основные функции

### 👤 Для пользователей
- 🛒 Корзина покупок с динамическим обновлением
- 🏪 Каталог магазинов с фильтрацией по категориям
- 🔍 Поиск товаров и магазинов
- 💳 Оформление заказа с выбором способа оплаты
- 📦 Отслеживание статуса заказа
- 💎 Система подписок (Стандарт, Плюс, Премиум)
- 👤 Личный кабинет с историей заказов

### 🔧 Для администраторов (`/admin`)
- 📊 Дашборд со статистикой
- 👥 Управление пользователями
- 🏢 Управление заявками партнеров
- 🚚 Управление курьерами
- 📦 Управление заказами с фильтрацией
- ⚙️ Настройки приложения (главный экран, уведомления)
- 📈 Аналитика и отчеты

### 🏪 Для партнеров (`/partner`)
- 🏪 Настройка магазина (название, обложка, описание)
- 📦 Управление товарами и категориями
- 🖼️ Загрузка изображений товаров
- 📊 Аналитика продаж
- 📦 Управление заказами и их статусами
- 💰 Статистика выручки

### 🚚 Для курьеров (`/courier`)
- 📍 Список активных заказов
- ✅ Управление статусами доставки
- 📊 Статистика смен
- 🗺️ Навигация до адреса
- ⏱️ Переключение статуса "на смене"

## 💳 Система оплаты

- **Банковская карта** - Ввод данных карты
- **СБП** - Быстрая оплата через банки
- **Подписки** - Отдельный поток оплаты для подписок
- **Страница подтверждения** - Разные экраны для заказов и подписок

## 🚀 Установка и запуск

1. **Клонировать репозиторий**
```bash
git clone https://github.com/kazimovnikita0208-create/LavkinKot-.git
cd LavkinKot-
```

2. **Установить зависимости**
```bash
npm install
```

3. **Запустить dev сервер**
```bash
npm run dev
```

4. **Открыть в браузере**
```
http://localhost:3000
```

## 📱 Основные маршруты

- `/` - Главная страница

## 💎 Особенности

### Подписки
Три тарифных плана:
- **Стандарт** - 5 доставок за 1499₽ (30 дней)
- **Плюс** - 7 доставок за 1699₽ (45 дней)
- **Премиум** - 10 доставок за 1999₽ (60 дней)

### Роли пользователей
- **Пользователь** - Покупки и подписки
- **Администратор** - Полное управление
- **Партнер** - Управление магазином
- **Курьер** - Доставка заказов
